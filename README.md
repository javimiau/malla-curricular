# Malla Curricular Interactiva - Ingeniería Civil Informática UBB 

Este es un proyecto personal diseñado para gestionar y visualizar el progreso académico en la carrera de **Ingeniería Civil en Informática**.

## Características
* **Visualización Compacta:** Diseño optimizado para ver los 10 semestres en una sola pantalla.
* **Persistencia de Datos:** El progreso se guarda automáticamente en el navegador (`LocalStorage`).
* **Lógica de Requisitos:** El sistema valida si cumples con los créditos y prerrequisitos antes de permitirte aprobar un ramo.

## Tecnologías utilizadas
* **HTML5 / CSS3:** Maquetación y estilos personalizados.
* **JavaScript (ES6+):** Lógica de negocio y manipulación del DOM.
* **JSON:** Estructura de datos para las asignaturas.

## Cómo usarlo
1.  **Aprobar:** Haz clic en una asignatura pendiente. Si cumples los requisitos, se marcará en verde.
2.  **Desmarcar:** Si te equivocaste, haz clic en la asignatura aprobada para volverla a estado pendiente.
3.  **Progreso:** Mira el contador superior para ver cuántos créditos SCT llevas acumulados.
